# OpenDotA

这是一个基于 [opendota.com](https://www.opendota.com/) 开放数据的在线 DotA
数据查询网站。网站使用了 [react-boilerplate](https://github.com/mxstbr/react-boilerplate)
模板基于 [React.js](https://facebook.github.io/react/) 相关技术开发实现。
